What is a front-end framework?

  • A package made up of a structure of files and folders of standardized code which can be used to support the development of websites, as a basis to start building a site.

(https://www.awwwards.com/what-are-frameworks-22-best-responsive-css-frameworks-for-web-design.html)

Why a front-end framework?

  • Provides a common structure so that developers don’t have to redo it from scratch and can reuse the code provided.
  • Allow us to cut out much of the work and save a lot of time.

(https://www.awwwards.com/what-are-frameworks-22-best-responsive-css-frameworks-for-web-design.html)
