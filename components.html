Icons.

Font Awesome are Eden's chosen icon set. They provide a scalable set of vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS. Just download the directoty and install by referenceing their CSS file within the <head> of your document.

Container.

The Eden container is used to hold elements for which you want to give some space to aorund the edegs. By housing these elements within a .container class, a large padding is given to the top and bottom with a repsonive margin that adjusts on either side in response to the window width. If no space is needed, a .wrapper class can be used.

Grid.

The grid is a vital component in developing mobile responsive websites. We provide a 12 column fluid grid taht shrinks as the viewport shrinks. The Eden grid system uses .row classes directly followed by .col classes. The size (how many columns they occupy) of each column can then be specified by giving it a class of that number. e.g. a column spanning 7 columns would have the class .seven.

Eden also allows users to specify common fractions indicating the amount of it's parent it spans. e.g. If you want a column to span a third of the width, give it a class of .third or if you want it to occupy five sixths, make it .five-sixths. Simple isn't it?

To alter the column layout resposnively at mobile or tablet size, just add a class of .tab-* or .mob-* respectively where * is the column size.

Typography.

Eden comes with Source Sans Pro as its default font family but can easily be overwritten by changing it for the .body class. All types have been set using rem and em units where the default :root size is 12pt.

Other basic type styles such as .uppercase, .lowercase as well as different forms of alignemnt are all available.

Nav.

The navigation bar is one of, if not the most complex and feature rich component of the framework. Both horizontal and vertical navigation bars are potential options as well as sidebars that can be pulled out.

You start of with a .nav class where colour themes can also be given ( .light & .dark ). Sizes can also be specified ( .skinny, .fat, .short & .tall ) which affects the height and width of the nav bars. Alignment and position changes can also be made. Adding the classes fixed to .nav will make it stick to the top of your viewport. Adding .centered and .padded can center the text in the menu and also pad the sides respectively.

Within the .nav class, seperate your menu items with a ul tag with a .nav-options class.

Hero.

Hero images are a very popular and modern trend of web design. They are generally the first thing you see on a webpage and are used to grab your attention.

The Eden hero comes in all the sizes, .short, .tall, .skinny and .fat. The deafult will take up the entire height of your viewport and the entire width of the parent element. By adding a .row element immediataely inside, you have the ability to position an element in 9 places covering left, right and center along the x-axis and top, center, bottom along the y. So if you want some text in the bottom-right you would simply give your .row, a class of .bottom-right. Simples no?

Hero.

Eden heroes also support colour overlays which are great when you are looking to match your brand's theme or even to cover up a pixelated image. Adding a .overlay class to your .hero will automatically apply the effect. You can then choose your own colour by overriding the .overlay:after property in your CSS or you can apply Eden's .dark or .light theme.

Modal.

Modals are pop-up dialog boxes that are a nice and tidy way to present iformation without the need to open another page. Just make a .modal div as the next sibling of the element that you want to activate it. This must then contain two immediate children being, .content and .close. Inside .content you can do whatever you like and style it however you want. .close however should contain what you want to use to close the modal.
